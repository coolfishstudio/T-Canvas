# 说明
参考了一些游戏的玩法和设计，准备做一个休闲一些的多人同时在线的游戏。游戏将采用node＋socket.io＋canvas制作而成pc游戏。
后期考虑增加移动端模式、pc打包以及跨屏互动模式。

## 形象
暂定为表情包，后期改为类似于超级玛丽的像素人物形象。

## 操作
- w: 上
- d: 右
- s: 下
- a: 左
- j: 跳
- k: 打
- enter: 聊天
- tab: 排名
- F1: 说明
